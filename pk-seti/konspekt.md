# Конспект по компьютерным сетям

Компьютерная сеть - это совокупность устройств, которые соединены каналами связи и обмениваются данными по установленным правилам.

## Основные понятия

| Понятие | Описание |
|---|---|
| Узел сети | Компьютер, сервер, маршрутизатор, принтер или другое сетевое устройство |
| IP-адрес | Логический адрес устройства в сети |
| MAC-адрес | Аппаратный адрес сетевого интерфейса |
| DNS | Система, которая сопоставляет доменные имена и IP-адреса |
| Маршрутизатор | Устройство, передающее пакеты между разными сетями |
| Коммутатор | Устройство, соединяющее узлы внутри локальной сети |

## Модель OSI

| Уровень | Назначение |
|---|---|
| 7. Прикладной | Сервисы для приложений: HTTP, FTP, SMTP |
| 6. Представления | Кодирование, сжатие и шифрование данных |
| 5. Сеансовый | Управление сеансами связи |
| 4. Транспортный | Доставка данных между приложениями: TCP, UDP |
| 3. Сетевой | Маршрутизация пакетов: IP |
| 2. Канальный | Передача кадров внутри локальной сети: Ethernet, Wi-Fi |
| 1. Физический | Кабели, радиосигналы и электрические параметры |

## Вывод

Для настройки и диагностики сети важно понимать адресацию, работу DNS, различие между локальной и глобальной сетью, а также базовые команды: `ipconfig`, `ping`, `tracert`, `nslookup`, `ssh`.
