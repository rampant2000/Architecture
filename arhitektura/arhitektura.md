# Краткий конспект по архитектуре вычислительных средств

Архитектура вычислительных средств описывает, из каких узлов состоит компьютерная система и как эти узлы взаимодействуют между собой при выполнении программы.

## Основные элементы

| Элемент | Назначение |
|---|---|
| Центральный процессор | Выполняет команды программы и управляет обменом данными |
| Арифметико-логическое устройство | Выполняет арифметические и логические операции |
| Устройство управления | Организует выборку, декодирование и выполнение команд |
| Оперативная память | Хранит исполняемые программы и временные данные |
| Постоянная память | Содержит данные, которые должны сохраняться без питания |
| Системная шина | Передает адреса, данные и управляющие сигналы между устройствами |

## Принцип фон Неймана

Классическая модель компьютера строится на нескольких идеях:

- программа и данные хранятся в одной памяти;
- команды выполняются последовательно;
- процессор получает команду из памяти, декодирует ее и выполняет;
- результаты операций записываются обратно в регистры или память.

## Память

Память компьютера имеет иерархическую структуру. Самые быстрые уровни находятся ближе к процессору: регистры, кэш L1/L2/L3, оперативная память. Более медленные уровни, например SSD или HDD, используются для долговременного хранения.

## Вывод

Понимание архитектуры вычислительных средств помогает оценивать производительность компьютера, выбирать компоненты для модернизации и объяснять, почему одна система работает быстрее другой.
